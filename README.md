# The Podcast Show London - Simplified Redesign

A clean, modern redesign of The Podcast Show London website focused on clarity and simplicity.

## Overview

This is a simplified alternative design for The Podcast Show London website, created to address concerns about clutter and improve user experience. The redesign focuses on:

- **Cleaner navigation** - Simplified menu structure
- **Better visual hierarchy** - Clear content organization
- **Reduced clutter** - Streamlined sections and information
- **Modern design** - Contemporary UI with improved readability
- **Responsive design** - Works seamlessly on all devices

## Key Improvements

### Navigation
- Single-level navigation menu (removed nested dropdowns)
- Clear call-to-action button
- Sticky header for easy access

### Hero Section
- Single, focused message
- Clear date and location
- Prominent CTAs

### Content Organization
- Consolidated information into clear sections
- Removed redundant content
- Better spacing and typography
- Simplified ticket pricing display

### Design Elements
- Modern color scheme
- Improved typography with Inter font
- Better use of white space
- Subtle animations and hover effects

## Customization

### Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #1a1a1a;
    --accent-color: #0066ff;
    /* ... */
}
```

### Content
All content is in `index.html` - simply edit the HTML to update text, links, and structure.
